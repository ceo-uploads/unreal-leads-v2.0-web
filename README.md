---

## ⚡ CORE CHARACTERISTICS

### 🎨 Elite Visual Engineering
- **Million-Dollar Aesthetic:** Ultra-dark mode UI with brand-gold accents and technical "Swiss-Design" typography.
- **Micro-Interactions:** Smooth Framer Motion animations, staggered entrances, and hover-responsive layouts.
- **Glassmorphism:** High-end backdrop blurs and translucent surface architectures.

### 📊 Elite Data Marketplace
- **Curated Datasets:** Access to highly filtered, verified leads across multiple industries (E-commerce, Real Estate, Local Business).
- **Subscription Architecture:** Tiered licensing (Starter, Pro, Elite, Lifetime) with instant activation upon payment approval.

### 🛡️ Secure Transaction Layer
- **Multi-Gateway Support:** Integrated manual payment flows for bKash, Nagad, Nsave (Ntag), and Binance Pay.
- **Proof-of-Transaction:** Robust verification system using Transaction IDs and sender account tracking.
- **WhatsApp Integration:** Automated delivery prompts and direct communication bridges.

### ⚙️ Command Control (Admin Panel)
- **Real-Time Monitoring:** Live transaction feeds and user activity streams.
- **One-Tap Approval:** Quick verification and license provisioning.
- **Identity Restoration:** Custom feature to restore inactive/closed packages to active status.
- **Software Access Management:** Creation and deletion of dedicated software-level credentials.

## 🏗️ INSTALLATION & SETUP

1. **Clone the Identity:**
   ```bash
   git clone [https://github.com/assassingamerbd4/unreal-leads.git](https://github.com/ceo-uploads/unreal-leads-v2.0-web)
   cd unreal-leads
   ```

2. **Initialize Dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env` file and populate your Firebase configuration:
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_DATABASE_URL=your_db_url
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   ```

4. **Launch Development Environment:**
   ```bash
   npm run dev
   ```
